define container

to define grid container, we need to add display property with value of grid inside the css-rule
Example:

#container_id{display:grid;}

and to define the grid columns & rows, we must input the properties grid-templet-columns & grid-template-rows with values:

  • fixed (px)
  • or relative (% em rem vh vw vmax vmin)
  • or fractions (fr) frome left spaces
  • or shrink (auto) to its original content size
and we can repeat them by the function:
  • repeat(times or auto-fill or auto-fit, value)
also we can give the values range by the function:
  • min-max(minimum Value, maximum Value)
Example:

#container_id{
  grid-template-columns:auto repeat(auto-fit,minmax(60px,2fr)) 3fr;
  grid-template-rows:repeat(auto-fill,minmax(auto, 20%)) 7%;
}

and for shorthand we can use just the property grid-template, with value of "rows values / columns vakues"
Example:

#container_id{
  grid-template:auto repeat(auto-fit,minmax(60px,2fr)) 3fr / repeat(auto-fill,minmax(auto, 20%)) 7%;
}

define items

to define how much grid cells items spanning, we can use the two properties grid-column & grid-row with value of:

  • "start horizontal line / end horizontal line"
  • "start vertical line / end vertical line"
Example:

#item_id{
  grid-column:1/4;
  grid-row:2/4;
}

or we can give them the value "span cells-number"
Example:

#item_id{
  grid-column:span 3;
  grid-row:span 2;
}

Note: span values make items flexible and not fixed.

adjust items positions

  1. to adjust all items position at ones, we can use the properties justify-items (main axis) & align-items (cross axis) with values of "start center end"
    Example:

    #container_id{
  justify-items:center;
  align-items:end;
}

  2. also we can adjust a single item position, by the propertys justify-self (main axis) & align-self (cross axis)
    Example:

    #item_id{
  justify-self:start;
  align-self:center;
}

adjust grid spacing

to make space between items, we can use the properties grid-column-gap & grid-row-gap with value of "px % em rem vh vw vmax vmin"
Example:

#container_id{
  grid-column-gap:5px;
  grid-row-gap:2%;
}

as shorthand we can use grid-gap, with value of "rows-gap columns-gap"
Example:

#container_id{
  grid-gap:2% 5px;
}

and we can assign the same value to the both
Example:

#contaner_id{
  grid-gap:2vmin;
}

define grid areas

  1. to define grid areas, we required to type the property grid-template-areas on container rule, with value of rows="cells or ." which "." mean empty cell
    Example:

    #container_id{
  grid-template-areas:
  "header header"
  "advert content"
  "   .   content"
  "footer footer"
}

  2. to assign areas to items, we should enter the property grid-area inside item rule, and give it value of "cell"
    Example:

    #footer_item_id{
  grid-area:footer;
}
